# LISTEN Time Clock

放課後等デイサービス LISTEN 向け  
出勤・退勤打刻アプリ

---

# 1. 目的

本システムは、職員の **出勤時刻** および **退勤時刻** をタブレットから簡単に記録することを目的とする。

本システムは勤務時間計算や勤務表作成を目的とせず、  
**タイムカードの代替として打刻ログを保存すること** を主目的とする。

---

# 2. システム概要

職員はタブレット画面から

- 自分の氏名を選択
- 出勤ボタン または 退勤ボタン を押す

ことで打刻を行う。

打刻データは Google スプレッドシートに保存する。

# 7. スプレッドシート構成

スプレッドシートには以下のシートを作成する。

---

## employees

従業員名簿

|列|項目|内容|
|---|---|---|
|A|employee_id|職員ID|
|B|name|氏名|
|C|active|表示対象|

### 例

|employee_id|name|active|
|---|---|---|
|E001|田中|TRUE|
|E002|佐藤|TRUE|
|E003|鈴木|FALSE|

active が TRUE の従業員のみ表示する。

---

## attendance_log

打刻ログ保存シート

**1打刻につき1行追加保存**

|列|項目|
|---|---|
|A|date|
|B|weekday|
|C|time|
|D|name|
|E|action|
|F|timestamp|

---

### 保存例

|date|weekday|time|name|action|timestamp|
|---|---|---|---|---|---|
|2026/03/10|火|08:31:12|田中|出勤|2026/03/10 08:31:12|
|2026/03/10|火|17:12:01|田中|退勤|2026/03/10 17:12:01|

---

## 退勤打刻

---

## 打刻ミス
