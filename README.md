# LISTEN Time Clock

放課後等デイサービス LISTEN 向け  
出勤・退勤打刻アプリ

---

# 1. 目的

本システムは、職員の **出勤時刻** および **退勤時刻** をタブレットから簡単に記録することを目的とする。

本システムは勤務時間計算や勤務表作成を目的とせず、  
**タイムカードの代替として打刻ログを保存すること** を主目的とする。

---

# 2. システム概要

職員はタブレット画面から

- 自分の氏名を選択
- 出勤ボタン または 退勤ボタン を押す

ことで打刻を行う。

打刻データは Google スプレッドシートに保存する。

---

# 3. システム構成

本システムは以下の3つで構成する。

## フロントエンド
GitHub Pages 上で公開する。

使用技術

- HTML
- CSS
- JavaScript

役割

- 打刻画面表示
- 従業員一覧表示
- 氏名選択
- 出勤・退勤ボタン
- 打刻結果表示

---

## バックエンド

Google Apps Script (GAS)

役割

- 従業員名簿取得
- 打刻データ受信
- スプレッドシート保存

---

## データ保存

Googleスプレッドシート

役割

- 従業員名簿管理
- 打刻ログ保存
